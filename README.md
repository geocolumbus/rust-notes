# Rust notes

Install Rust using ```rustup``` - [instructions](https://www.rust-lang.org/tools/install).

```
rustc --version
rustc 1.35.0 (3c235d560 2019-05-20)

rustup --version
rustup 1.18.3 (435397f48 2019-05-22)
```

## Usage
```
git clone https://github.com/geocolumbus/rust-notes.git
cd rust-notes
rustc main.rs
./main
```
